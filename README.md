# Estudos e Projetos em C

Este repositório reúne meus estudos, exercícios e projetos desenvolvidos na linguagem **C**, com o objetivo de fortalecer a lógica de programação e compreender os fundamentos da programação estruturada.

Durante o desenvolvimento das atividades, busco aplicar boas práticas de codificação, resolver problemas utilizando algoritmos eficientes e construir uma base sólida para o aprendizado de linguagens e tecnologias mais avançadas.

## Conteúdos estudados

* Sintaxe da linguagem C;
* Variáveis e tipos de dados;
* Operadores aritméticos, relacionais e lógicos;
* Estruturas de decisão (`if`, `else`, `switch`);
* Estruturas de repetição (`for`, `while` e `do-while`);
* Funções;
* Vetores e matrizes;
* Strings (`string.h`);
* Structs;
* Ponteiros;
* Manipulação de arquivos (`FILE`);
* Modularização de código;
* Desenvolvimento de aplicações em modo console.

## Projetos desenvolvidos

* Sistema de Gerenciamento de Biblioteca;
* Exercícios de lógica de programação;
* Algoritmos para resolução de problemas;
* Projetos práticos para fixação dos conceitos da linguagem.

## Objetivo do repositório

Este repositório faz parte da minha jornada de formação em Engenharia de Software e documenta minha evolução no aprendizado da linguagem C. Os projetos aqui desenvolvidos servem como base para o estudo de estruturas de dados, algoritmos e programação de baixo nível, além de fortalecer os fundamentos necessários para o desenvolvimento em linguagens como Java, C++ e Python.

## Tecnologias utilizadas

* Linguagem C
* GCC (GNU Compiler Collection)
* Dev-C++ / Code::Blocks / Visual Studio Code (dependendo do ambiente de desenvolvimento)
* Git e GitHub
